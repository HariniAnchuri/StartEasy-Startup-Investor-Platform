# StartEasy – Startup & Investor Collaboration Platform

## About the Project

StartEasy is a web-based platform developed to bridge the gap between entrepreneurs and investors. The platform allows startup founders to present their business ideas while enabling investors to explore and evaluate potential investment opportunities.

The project provides a centralized environment where users can register, log in, submit startup ideas, and manage information through a user-friendly interface.

## Features

- User Registration and Login System
- Startup Idea Submission
- Investor Dashboard
- Idea Management System
- Database Connectivity using MySQL
- Responsive and Interactive User Interface
- Secure Data Storage

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP
- phpMyAdmin

## Project Structure

- HTML pages for frontend design
- CSS for styling and responsiveness
- PHP scripts for backend processing
- MySQL database for storing user and startup information
- Image assets for user interface design

## How to Run

1. Install XAMPP.
2. Start Apache and MySQL services.
3. Import the database into phpMyAdmin.
4. Copy the project folder into the `htdocs` directory.
5. Open the project in a web browser using localhost.
6. Register or log in to access the platform features.
